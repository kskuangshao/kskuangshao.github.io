hue安装部署

hue安装部署

    需求：统计某些指标、导数
数据：Hive/SparkSQL/Impala/RDBMS

实现方式一：自己动手统计
    登录到Hive所在机器
    编写Hive QL语句，得到结果
    结果放到Excel：柱状图、折线图、饼图等等

实现方式二：通过类似的数据平台
    需求提给平台组
    等待平台组去处理
    拿到结果，自己展示

    ==>

数据平台上如果能够提供/开放出来类似于UI的界面
在UI界面上输入SQL，拿到结果并直接在平台上进行图形化的展示

开源的框架是否有类似的支持的：HUE Zeppelin
是在日常的工作中是非常实用的
hue的官方网站 https://gethue.com/
前置要求：  Python2.7     MySQL       Java
1.安装依赖

sudo yum install -y ant asciidoc cyrus-sasl-devel cyrus-sasl-gssapi cyrus-sasl-plain gcc gcc-c++ krb5-devel libffi-devel libxml2-devel libxslt-devel make  mysql mysql-devel openldap-devel python-devel sqlite-devel gmp-devel

2.下载HUE	
		http://archive.cloudera.com/cdh5/cdh/5/     cdh5.16.2

		wget http://archive.cloudera.com/cdh5/cdh/5/hue-3.9.0-cdh5.16.2.tar.gz
3.解压HUE到~/app下
	切换到~/app/hue-3.9.0-cdh5.16.2
	make apps
4.HUE配置
    $HUE_HOME/desktop/conf/hue.ini
    [desktop]
            ##这个随便写，越长越好
			secret_key=qwert1234534566
			http_host=JD
			http_port=38888
			time_zone=Asia/Shanghai
			default_user=hadoop
			default_hdfs_superuser=hadoop
			localhost=JD
			
	        resourcemanager_host=JD
	        resourcemanager_port=8032
	        resourcemanager_api_url=http://JD:8088
	        proxy_api_url=http://JD:8088
	        history_server_api_url=http://JD:19888
5.HUE整合HDFS
	core-site.xml
		<property>
			<name>hadoop.proxyuser.hadoop.hosts</name>
			<value>*</value>
		</property>
		<property>
			<name>hadoop.proxyuser.hadoop.groups</name>
			<value>*</value>
		</property>
		
	hdfs-site.xml
		<property>
			<name>dfs.webhdfs.enabled</name>
			<value>true</value>
		</property>
	修改完Hadoop相关的配置之后，需要重启才生效
6.HUE整合MR/YARN
	你需要启动  mr-jobhistory-daemon.sh 
	mapred-site.xml
		<property>
			<name>mapreduce.jobhistory.address</name>
			<value>JD:10020</value>
		</property>

		<property>
			<name>mapreduce.jobhistory.webapp.address</name>
			<value>JD:19888</value>
		</property>

		<property>
			<name>mapreduce.jobhistory.done-dir</name>
			<value>/history/done</value>
		</property>

		<property>
			<name>mapreduce.jobhistory.intermediate-done-dir</name>
			<value>/history/done_intermediate</value>
		</property>
7.HUE整合MySQL
	访问MySQL的数据
	搜/librdbms
	nice_name="kskuangshao DB"
	name=hivedb
	engine=mysql
	host=JD
	port=3306
	user=root
	password=123456
不能多人并发访问mysql的问题：（在mysql里创建hue库。。。。官网有叙述，这里不记录）
8.HUE整合Hive
	一定要先启动hiveserver2	
	/beeswax
	hive_server_host=JD
	hive_server_port=10000
	hive_conf_dir=/var/local/hive/conf
9.启动HUE
		hue目录下
		build/env/bin/supervisor
hue
